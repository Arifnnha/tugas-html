Apa Itu HTML dan CSS?

HTML (HyperText Markup Language) adalah bahasa yang digunakan untuk membuat struktur dan konten sebuah halaman web. HTML berfungsi untuk menentukan elemen-elemen seperti teks, gambar, tautan, dan lainnya dalam sebuah dokumen.

Contoh Struktur Dasar HTML Dengan External CSS

CSS (Cascading Style Sheets) adalah bahasa yang digunakan untuk mengatur tampilan dan tata letak dari elemen-elemen HTML. Dengan CSS, kita bisa mengubah warna, ukuran, posisi, dan aspek visual lainnya dari konten yang telah dibuat dengan HTML.
